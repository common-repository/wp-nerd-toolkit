== Description ==

Mit dem [WP]:NERD Toolkit hast du starke leistungssteigernde Tools mit an Board und sparst dir weitere Plugins.
Was ist alles dabei?

*   One-Click SSL Tool
*   Backup Funktion
*   Minify Tool
*   Entferne Query Strings
*   Deaktiviere die WordPress Emojis
*   Entferne deine WordPress Version aus dem Quellcode
*	Füge den Google Analytics Tracking-Code ein
*	Füge den Facebook Pixel ein

Weitere Funktionen sind in Planung und werden zeitnahe umgesetzt.
Wenn du Ideen hast oder ein Feature welches unbedingt dabei sein sollte, sag uns bescheid!

NEU! Seit Version 1.1 gibt es nun eine Backup-Funktion! Erstelle ein komplettes Backup auf deinem Server oder via FTP auf einem Backup-Server.

Support:
[Tobier Webentwicklung](https://tobier.de)

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the [WP]:NERD Toolkit screen to configure the plugin
